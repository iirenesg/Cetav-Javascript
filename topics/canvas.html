Canvas

Canvas setup

      
        <canvas id="canvas" width="1280" height="720">
          // You can use this inner content to show a fallback graphic like an image or a video
        </canvas>

        const canvas = document.getElementById('canvas');
        const ctx = canvas.getContext('2d');

The context is the interface for all the canvas methods. There's two main contexts: '2d' and 'webGL'. Here's the documentation for the 2D Context

Drawing basics

      
        ctx.fillStyle = 'black';
        ctx.fillRect(75, 75, 150, 150);

        ctx.fillStyle = 'white';
        ctx.fillRect(150, 75, 150, 150);

Rectangles

      
        ctx.strokeStyle = 'white';
        ctx.lineWidth = 15;
        ctx.strokeRect(75, 75, 150, 150);

Circles

      
        ctx.beginPath();
        ctx.strokeStyle = 'black';
        ctx.arc(canvas.width / 2, canvas.height / 2, 100, 0, 2 * Math.PI);
        ctx.stroke();
Angles by default use radians, which is the angle measurement when the arc is the same length as the radius of the circle.

Paths

      
        ctx.beginPath();
        ctx.moveTo(225, 225);
        ctx.lineTo(225, 45);
        ctx.lineTo(45, 225);
        ctx.closePath();
        ctx.stroke();

Animation

We can animate using serveral methods:

  • setTimeout
  • setInterval
  • requestAnimationFrame

RequestAnimationFrame is the recommended way to handle animations in the browser because it lets the browser execute the code according to its own framerate.

Browsers do not update their display while a JavaScript program is running, nor do they allow any interaction with the page, so using timeouts or intervals can cause performance issues.

You can read the requestAnimationFrame documentation here

Animation

      
        let xPosition = 0;

        function draw() {
          ctx.clearRect(0, 0, canvas.width, canvas.height);
          ctx.fillStyle = 'black';
          ctx.fillRect(xPosition, 25, 50, 50);

          if (xPosition > canvas.width) {
            xPosition = 0;
          }

          xPosition += 3;
          window.requestAnimationFrame(draw);
        }

        window.requestAnimationFrame(draw);
